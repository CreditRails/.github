# Roadmap

## Phase 1 — Scoring Foundation ✅ (Testnet)
- Stellar Horizon indexer live on testnet
- Behavioral scoring engine producing scores from wallet history
- Credit score dashboard for users
- Internal API returning score and risk tier

## Phase 2 — Credential Issuance 🔄 (In Progress)
- W3C DID credential issuance
- User-owned credential containing score and eligibility
- Credential verification endpoint for lenders
- Share credential flow in dashboard

## Phase 3 — Blend Integration 📋 (Planned)
- CreditRails risk API connected to Blend lending pools
- Lenders query borrower eligibility before loan issuance
- Undercollateralized loan flow tested end-to-end on testnet
- Dynamic loan terms based on score tier

## Phase 4 — Signal Expansion 📋 (Planned)
- Blockroll payroll signals
- Sava savings signals
- Fiatsend + VANK remittance signals
- Weighted model updated with new signal types

## Phase 5 — Mainnet + SDK 📋 (Planned)
- Indexer and scoring on mainnet
- Real users generating real credit profiles
- Open-source SDK published for ecosystem integrations
- At least one active lending integration with real loan volume
